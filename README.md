# Material UI Replica

## Overview
This project is a frontend application that replicates a Material UI-based design using Angular.

It focuses on building reusable UI components and implementing responsive layouts.

## Tech Stack
- Angular
- TypeScript
- HTML, SCSS

## Features
- Responsive UI design
- Component-based architecture
- Clean layout inspired by Material UI

## Live Demo
https://ichigo-psi.vercel.app/

## How to Run

1. Clone the repository
2. Install dependencies:
   npm install
3. Run the app:
   ng serve
4. Open http://localhost:4200
